# \<kendo:diagram-connectionDefaults-hover\>

Defines the hover configuration.

#### Example
    <kendo:diagram-connectionDefaults>
        <kendo:diagram-connectionDefaults-hover></kendo:diagram-connectionDefaults-hover>
    </kendo:diagram-connectionDefaults>
